## EXPLAIN

### CLI 기초 — 5가지 필수 명령어

| 명령어 | 비유 | 실제 동작 |
|--------|------|----------|
| `cd` | Finder에서 폴더 더블클릭 | 디렉토리 이동 |
| `ls` | 폴더 열어서 내용 보기 | 파일/폴더 목록 표시 |
| `pwd` | "지금 어디에 있지?" | 현재 경로 출력 |
| `cat` | 파일 미리보기 | 파일 내용 터미널에 출력 |
| `open` | 파일 더블클릭 | 기본 앱으로 열기 |

**기억할 필요 없습니다.** Claude에게 "현재 폴더 알려줘"라고 하면 `pwd`를 실행합니다.

### Git 기초 — 게임 세이브

```
git commit = 게임 세이브 (현재 상태 저장)
git push   = 클라우드에 업로드
git pull   = 클라우드에서 다운로드
git log    = 세이브 기록 확인
```

**Before Git:**
```
보고서.docx
보고서_수정.docx
보고서_최종.docx
보고서_최종_찐최종.docx
보고서_진짜최종_이게마지막.docx
```

**After Git:**
```
commit 1: "초안 작성"
commit 2: "피드백 반영"
commit 3: "최종 검토 완료"
→ 깔끔하게 버전 관리
```

### GitHub 기초

- **Repository (저장소)**: 프로젝트의 모든 파일과 히스토리가 저장되는 곳
- **Pull Request (PR)**: "이 변경 사항 검토 후 합쳐주세요" 요청
  - 코드 변경 → PR 생성 → 팀원 검토 → 승인 → 병합
