# 레볼루션 6: 리서치와 학습의 결합

## 목표
self-learning 스킬과 domain-research 스킬을 결합하여 자기주도 심화 학습 방법을 익힙니다.

## 실행 프로토콜

### Phase A (설명 + 실습)

참조 파일 `references/rev6-research-integration.md`를 읽으세요.

#### 학습의 다음 단계: 리서치

지금까지 Claude Code의 기본 기능을 학습했습니다.
이제 이 도구를 활용해서 **실제 주제를 깊이 연구**하는 방법을 배웁니다.

#### domain-research 스킬 소개

이 플러그인에는 `domain-research` 스킬이 포함되어 있습니다:

```
Step 0: 대화형 의도 분석 → 연구 맥락 구축
Step 1: 핵심 질문 생성 (5개)
Step 2: 연구 공백 식별
Step 3: 개별 소스 분석
Step 4: 다중 소스 통합
Step 5: 실행 계획 수립
Final: 종합 가이드 생성
```

#### 자기주도 학습 워크플로우

1. **관심 주제 선정**
   - 예: "AI를 활용한 업무 자동화"
   - 예: "우리 팀에 맞는 MCP 서버 구축"

2. **domain-research로 체계적 탐구**
   ```
   /research "Claude Code를 활용한 팀 생산성 향상"
   ```

3. **학습한 기능 활용**
   - Skill로 반복 작업 자동화
   - MCP로 외부 데이터 연결
   - Hook으로 워크플로우 자동화

4. **결과물을 Skill로 정리**
   - 학습한 내용을 새로운 Skill로 만들기
   - 팀에 Plugin으로 공유

#### 실습

다음 중 하나를 시도해보세요:
1. `domain-research` 스킬로 관심 주제 리서치 시작하기
2. 학습한 내용을 정리하는 "학습 노트" 스킬 만들기
3. 자신의 업무에 적용할 수 있는 자동화 스킬 구상하기

마무리 멘트:
"👆 위 내용을 직접 시도해보세요. 완료되면 '완료'라고 입력해주세요."

**여기서 STOP.**
