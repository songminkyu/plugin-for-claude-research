참조 파일 `references/block4-basics.md`를 읽으세요.

#### CLI 기초 — 5가지 필수 명령어

| 명령어 | 비유 | 설명 |
|--------|------|------|
| `cd` | 폴더 더블클릭 | 디렉토리 이동 |
| `ls` | 폴더 열기 | 파일 목록 보기 |
| `pwd` | "지금 어디야?" | 현재 위치 확인 |
| `cat` | 파일 미리보기 | 파일 내용 출력 |
| `open` | 더블클릭 열기 | 파일/폴더 열기 |

#### Git 기초 — 게임 세이브

- `git commit` = 게임 세이브 (현재 상태 저장)
- `git push` = 클라우드 업로드
- `git pull` = 클라우드 다운로드
- 더 이상 `보고서_최종_찐최종.docx` 안 해도 됨

#### GitHub 기초

- Pull Request = "이 변경 좀 검토해주세요"
- Repository = 프로젝트 저장소
